How Background Learning Works
  1. When users transcribe audio on the main page, samples are silently queued
  2. Admin can review queued samples and trigger background training
  3. Model trains on these samples with low learning rate (safer)
  4. Trained samples are marked as learned and removed from queue
  5. Model improvements are saved automatically
  6. Process repeats continuously to improve accuracy
